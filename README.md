## 🔥 परिचय

**BUT Network** बिटकॉइन का क्वांटम-प्रतिरोधी, मोबाइल-फर्स्ट, और 10-Layer Security वाला विकल्प है।

### खासियत:
- 🔒 **10 Security Layers** — दुनिया का सबसे सुरक्षित ब्लॉकचेन
- 🔮 **512-bit & 1024-bit** क्वांटम-प्रतिरोधी CKS क्रिप्टोग्राफ़ी
- 🔑 **MPC Key Sharding** (3-of-5) — Key कभी एक जगह नहीं
- 🙈 **Zero-Knowledge Proofs** — बिना details दिखाए verification
- 🤖 **AI Shield** — Real-time threat detection
- 🔐 **Post-Quantum Signatures** — Dilithium-style lattice-based
- 📱 **Mobile Mining** — Proof-of-Connection, ज़ीरो बिजली
- 👁️ **BUT-S/BUT-V** डुअल की (गोपनीयता + ऑडिटेबल)
- 🌐 **but://username** — UPI जैसी सरलता
- 🆓 **Free Token Launch** — कोई भी अपना टोकन बनाए

## 🛡️ 10 Security Layers

| # | Code | Layer | Language |
|---|------|-------|----------|
| 1 | CKS | 512/1024-bit Base Crypto | Rust |
| 2 | VLT | MPC Key Sharding (3-of-5) | C++ |
| 3 | ZKF | Zero-Knowledge Proofs | C++ |
| 4 | AIS | AI Shield (Behavior Analysis) | C++ |
| 5 | QSH | Post-Quantum Signatures | Rust |
| 6 | TEX | Trusted Execution Environment | C++ |
| 7 | TLK | Time-lock Encryption | C++ |
| 8 | DID | Decentralized Identity | C++ |
| 9 | AIR | Air-gapped Signing | C++ |
| 10 | QRN | Quantum Random Numbers | Rust |

## 🏗️ Project Structure
Butcoin/
├── 🔐 but-crypto/          ← Rust: CKS-512, CKS-1024
├── 🌐 but-p2p/             ← Go: P2P + Noise + Shield
├── 🧱 but-core/            ← C++: Blockchain Engine
├── ⛏️ but-consensus/       ← C++: Proof-of-Connection
├── 📱 but-wallet/          ← Flutter: Mobile Wallet
│
├── 🔑 but-vault/           ← C++: VLT (MPC Key Sharding)
├── 🙈 but-zkp/             ← C++: ZKF (Zero-Knowledge)
├── 🤖 but-ai/              ← C++: AIS (AI Shield)
├── 🔮 but-quantum/         ← Rust: QSH (Quantum Sigs)
├── 🔒 but-tee/             ← C++: TEX (Trusted Execution)
├── ⏰ but-time/            ← C++: TLK (Time-lock)
├── 🆔 but-did/             ← C++: DID (Identity)
├── 📴 but-airgap/          ← C++: AIR (Air-gapped)
└── 🎲 but-random/          ← Rust: QRN (Quantum Random)
